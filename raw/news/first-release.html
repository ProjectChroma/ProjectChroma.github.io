Today, Chroma will be publicly available to play for the first time. The core of the game's systems, a handful of interesting tech-demo levels, and a basic launcher are now publicly available to download and play.

Chroma is only 23 days old - the first lines of code were committed on May 3, 2016, and has grown steadily ever since. The two repositories combined contain almost 2,500 lines of code and average eight commits a week.

You can download the Launcher here. The source code for the launcher is on Github, at ProjectChroma/Launcher.
